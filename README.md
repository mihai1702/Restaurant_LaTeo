## Descriere
**Restaurant_LaTeo** este un website de prezentare realizat cu **HTML, CSS, JavaScript, PHP, AJAX**, având urmatoarele functii:
- Homepage modern
- Vizualizare meniu
- Contact
- Pagina administrare pentru adaugarea, afisarea, modificarea si stergerea elementelor din meniu, sortare pe categorii si administrarea rezervarilor.
- Pagini statice pentru prezentare
- Meniu cu imagini și descrieri
- Pagini Web Responsive


**Vor urma implementate urmatoarele functii**:
- Posibilitatea de a efectua o rezervare
